Java 高级 - 反射

什么是反射

在运行状态中，对于任意一个类，都能够知道这个类的所有属性和方法；对于任意一个对象，都能够调用它的任意一个方法和属性；这种动态获取的信息以及动态调用对象的方法的功能称为java语言的反射机制。

获取Class对象

有三种方式：

User user = new User();
Class testClass1 = user.getClass();
System.out.println(testClass1.getName());

Class testClass2 = User.class;
System.out.println(testClass2.getName());

try {
    Class testClass3 = Class.forName("com.company.User");
    System.out.println(testClass3.getName());
} catch (ClassNotFoundException e) {
    e.printStackTrace();
}

输出结果为：com.company.User

其中第三种方法必须为完整的包名加类名。

获取构造器

公有构造方法

Constructor[] array = testClass1.getConstructors();

这样就获得一个构造方法的数组。而获取所有的构造方法可以使用：

array = testClass1.getDeclaredConstructors();

另外可以指定所获取的构造器的参数：

Constructor constructor = testClass1.getConstructor(null);	//获取公有无参构造器

constructor = testClass1.getDeclaredConstructor(String.class);	//获取有一个String参数的构造器

也可以直接调用获取到的构造方法：

Object obj = constructor.newInstance();	//无参
User user1 = (User) obj;
user1.setUsername("test");

Object obj = constructor.newInstance("test");	//有一个String参数
User user1 = (User) obj;
System.out.println(user1.getUsername());

在使用私有构造器时需要使用

constructor.setAccessible(true);

来进行强制访问。

获取字段

与获取构造器类似。

Field[] fields = testClass1.getFields();	//公有字段
fields = testClass1.getDeclaredFields();	//所有字段

公有字段的引用：

Field field = testClass1.getField("ID");
obj = testClass1.getConstructor().newInstance();
field.set(obj,"12345");
user = (User) obj;
System.out.println(user.ID);

这样就将就将"12345"传给了user对象的username中

所有字段（包括私有字段的引用）：

obj = testClass1.getConstructor().newInstance();
field = testClass1.getDeclaredField("username");
field.setAccessible(true);  //强制调用
field.set(obj,"testUsername");
user = (User) obj;
System.out.println(user.getUsername());

注意field.setAccessible(true);这一行代码强制调用私有字段，调用公有字段时不需要。

获取方法

Method[] methods = testClass1.getMethods(); //所有的共有方法
methods = testClass1.getDeclaredMethods();	//所有的自己声明的方法（包括私有方法）

调用获取到的方法：

公有方法：

Method method = testClass1.getMethod("show2", String.class);
obj = testClass1.getConstructor().newInstance();
method.invoke(obj,"test");

自己声明的方法（包括私有方法）：

method = testClass1.getDeclaredMethod("show1", String.class);
method.setAccessible(true);	//进行强制引用，调用私有方法时需要
method.invoke(obj,"test");

可以通过反射越过泛型：

List<String> testList = new ArrayList<>();
testList.add("test");
testClass1 = testList.getClass();
method = testClass1.getMethod("add", Object.class);
method.invoke(testList,123);
System.out.println(testList);

这样就可以添加Integer型数据。
