预备知识

第二个项目需要我们实现一个基于磁盘的哈希表，需要快速搜索数据而不需要遍历每一条数据。

需要使用可扩展散列方案实现散列表。

该索引包含一个目录页面，其中包含指向存储桶页面的指针。该表将通过项目#1 中的缓冲池访问页面。该表包含一个目录页面，其中存储了表和存储桶的所有元数据。哈希表需要支持完整/空存储桶的存储桶拆分/合并，以及全局深度必须更改时的目录扩展/收缩。

静态hash和动态hash

静态散列中存储桶的数量是确定的，也就是不管向其中插入多少数据，桶的数量总是不变的，通过hash函数计算得到的索引值也是不变的。如果桶的规模过小，当插入的数据过多时，每个桶中的数据就会很多，就会导致查找的性能大幅下降；如果桶的规模过大，那么就会导致有很大一部分空间被无效地占用，造成资源的浪费。

动态散列中存储桶的数量是不确定的，也允许根据需要改变hash函数，当插入的数据变多或者删除数据后，它会相应的改变存储桶的数量来对空间占用和性能上做一个平衡。它的两种常见的实现方式分别为可扩展动态散列和线性散列。在这个项目中，我们需要实现的就是一个可扩展动态散列。

可扩展动态散列

在上面这张图中

  • 参数k：最大位数
  • 参数i：全局深度（global_depth)
  • 参数n：目前最大桶的个数
  • 而每个桶又有其对应的local_depth,代表使用了几位数字

当插入二进制1110这个数时，我们只根据这个数的前i位来确定索引，发现这个桶中还没有满，所以可以插入

当再插入二进制1010时，根据其前i位来确定桶的索引值，发现桶已经满了，之后对这个桶进行分裂，然后对同种的元素和这个将要插入的元素重新进行hash运算，计算出新的索引值，将对应的元素放入新的桶中。

此后每次桶满后都会将这个桶分裂，每次分裂时都会比较local_depth和global_depth，如果这两个值相等的话，就需要增加global_depth。

这个方法获取全局和本地深度的掩码，取出对应的二进制位数

void HashTableDirectoryPage::IncrGlobalDepth() {
    int org_num = Size();
    for (int org_index = 0, new_index = org_num; org_index < org_num; new_index++, org_index++) {
        bucket_page_ids_[new_index] = bucket_page_ids_[org_index];
        local_depths_[new_index] = local_depths_[org_index];
    }
    global_depth_++;
}
void HashTableDirectoryPage::IncrLocalDepth(uint32_t bucket_idx) {
        local_depths_[bucket_idx]++;
}
void HashTableDirectoryPage::DecrLocalDepth(uint32_t bucket_idx) {
    local_depths_[bucket_idx]--;
}
void HashTableDirectoryPage::DecrGlobalDepth() { global_depth_--; }

这些函数用于改变全局深度和本地深度，当增加全局深度时也就增加了桶的个数，需要给新的桶赋予一些信息

uint32_t HashTableDirectoryPage::Size() {
    return 1 << global_depth_;
}

这个函数用于返回最大的桶的个数

bool HashTableDirectoryPage::CanShrink() {
    for (uint32_t i = 0; i < Size(); i++) {
        if (local_depths_[i] >= global_depth_) {
            return false;
        }
    }
    return true;
}

用于判断是否能够对全局深度也就是桶的最大个数进行压缩，当删除完某个桶中的元素后需要合并桶时需要判断全局深度能否进行收缩，当每个桶的本地深度都小于全局深度时就可以对全局深度进行压缩。

桶

  • occupied_ :如果 array_ 的第 i个索引曾经被占用，则 occupied_的第i位为 1。
  • readable_ :如果array_ 的第 i个索引持有一个可读值，readable_ 的第i位为 1。
  • array_ ：保存键值对的数组。

在这个桶的页面中我们需要实现添加，查找和删除的操作

//获取某个键对应的值
bool HASH_TABLE_BUCKET_TYPE::GetValue(KeyType key, KeyComparator cmp, std::vector<ValueType> *result) {
  bool res = false;
  for (size_t i = 0; i < BUCKET_ARRAY_SIZE; i++) {
      if (IsReadable(i) && cmp(array_[i].first, key) == 0) {

首先需要桶的这个位置是否可读，如果可读的话在使用特定的方法判断这两个键是否相等，如果相等的话就将结果添加到对应的结果容器中。

template <typename KeyType, typename ValueType, typename KeyComparator>
bool HASH_TABLE_BUCKET_TYPE::Insert(KeyType key, ValueType value, KeyComparator cmp) {
    size_t available = BUCKET_ARRAY_SIZE;
    for (size_t i = 0; i < BUCKET_ARRAY_SIZE; i++) {
        if (IsReadable(i)) {
            if (cmp(key, array_[i].first) == 0 && value == array_[i].second) {
                return false;
            }
        } else if (available == BUCKET_ARRAY_SIZE){
            available = i;
            break;
        }
    }
    if (available == BUCKET_ARRAY_SIZE) {
        return false;
    }
    array_[available] = MappingType(key, value);
    SetOccupied(available);
    SetReadable(available);
    return true;
}

这个函数用于向桶中插入新的数据，如果这个数据已经存在的话，那么就不插入，返回false，如果不存在的话就找到一个空闲的位置将这个数据插入到桶中并返回true。

template <typename KeyType, typename ValueType, typename KeyComparator>
bool HASH_TABLE_BUCKET_TYPE::Remove(KeyType key, ValueType value, KeyComparator cmp) {
    for (size_t i = 0; i < BUCKET_ARRAY_SIZE; i++) {
        if (IsReadable(i)) {
            if (cmp(key, array_[i].first) == 0 && value == array_[i].second) {
                RemoveAt(i);
                return true;
            }
        }
    }
    return false;
}

这个函数用于移除桶中的某个数据，如果键和值都和提供的键值相等的话就将这个数据从桶中删除。

template <typename KeyType, typename ValueType, typename KeyComparator>
bool HASH_TABLE_BUCKET_TYPE::IsOccupied(uint32_t bucket_idx) const {
    uint8_t value = static_cast<uint8_t>(occupied_[bucket_idx / 8]);
    return (value & (1 << (bucket_idx % 8))) > 0;
}

这个函数根据文档说明只需要判断对应位数的值是否为1即可

template <typename KeyType, typename ValueType, typename KeyComparator>
bool HASH_TABLE_BUCKET_TYPE::IsFull() {
    uint8_t sample = 255;
    size_t num = BUCKET_ARRAY_SIZE / 8;
    for (size_t i = 0; i < num; i++) {
        if ((sample & static_cast<uint8_t>(readable_[i])) != sample) {
            return false;
        }
    }
    size_t num_remain = BUCKET_ARRAY_SIZE % 8;
    if (num_remain > 0) {
        uint8_t value = static_cast<uint8_t>(readable_[num]);
        for (size_t i = 0; i < num_remain; i++) {
            if ((value & 1) == 0) {
                return false;
            }
            value >>= 1;
        }
    }
    return true;
}

这个函数用于判断是否这个桶已经满了，只需要判断readable数组中的每一位是否都被占用即可，需要注意的是除了整除的部分还有余数的部分也需要判断。

template <typename KeyType, typename ValueType, typename KeyComparator>
uint32_t HASH_TABLE_BUCKET_TYPE::NumReadable() {
    uint32_t num = 0;
    size_t r_num = BUCKET_ARRAY_SIZE / 8;
    for (size_t i = 0; i < r_num; i++) {
        uint8_t value = static_cast<uint8_t>(readable_[i]);
        for (size_t j = 0; j < 8; j++) {
            if ((value & 1) == 1) {
                num++;
            }
            value >>= 1;
        }
    }
    size_t r_num_remain = BUCKET_ARRAY_SIZE % 8;
    if (r_num_remain > 0) {
        uint8_t value = static_cast<uint8_t>(readable_[num]);
        for (size_t i = 0; i < r_num_remain; i++) {
            if ((value & 1) == 1) {
                num++;
            }
            value >>= 1;
        }
    }
    return num;
}

这个函数和IsFull函数类似，只需要判断对应位数的数字即可。

template <typename KeyType, typename ValueType, typename KeyComparator>
HashTableDirectoryPage *HASH_TABLE_TYPE::FetchDirectoryPage() {
  HashTableDirectoryPage *dir_page;
  if (directory_page_id_ == INVALID_PAGE_ID) {
      page_id_t new_dir_id;
      Page * page = buffer_pool_manager_->NewPage(&new_dir_id);
      dir_page = reinterpret_cast<HashTableDirectoryPage *>(page->GetData());
      dir_page->SetPageId(new_dir_id);
      directory_page_id_ = new_dir_id;
      page_id_t new_bucket_id;
      page = nullptr;
      page = buffer_pool_manager_->NewPage(&new_bucket_id);
      dir_page->SetBucketPageId(0, new_bucket_id);
      buffer_pool_manager_->UnpinPage(new_bucket_id, true);
      buffer_pool_manager_->UnpinPage(new_dir_id, true);
  }
  Page * page = buffer_pool_manager_->FetchPage(directory_page_id_);
  dir_page = reinterpret_cast<HashTableDirectoryPage *>(page->GetData());
  return dir_page;
}

这个函数用于获取目录页面，如果目录页面不存在的话就新建一个目录页面，并给这个页面赋一些初值。

用于查找给出的键对应的值。

template <typename KeyType, typename ValueType, typename KeyComparator>
bool HASH_TABLE_TYPE::Insert(Transaction *transaction, const KeyType &key, const ValueType &value) {
    HashTableDirectoryPage * dir_page = FetchDirectoryPage();
    page_id_t bucket_id = KeyToPageId(key, dir_page);
    HASH_TABLE_BUCKET_TYPE * bucket = FetchBucketPage(bucket_id);
    if (!bucket->IsFull()) {
        bool res = bucket->Insert(key, value, comparator_);
        buffer_pool_manager_->UnpinPage(bucket_id, true);
        buffer_pool_manager_->UnpinPage(dir_page->GetPageId(), false);
        return res;
    }
    buffer_pool_manager_->UnpinPage(dir_page->GetPageId(), false);
    buffer_pool_manager_->UnpinPage(bucket_id, false);
    return SplitInsert(transaction, key, value);
}

用于向给出的key对应的桶中插入数据，如果桶没有满则直接插入，如果桶满了，那么就需要将这个桶进行分裂，然后对同种的元素重新进行哈希运算，重新放入对应的桶中。

template <typename KeyType, typename ValueType, typename KeyComparator>
bool HASH_TABLE_TYPE::SplitInsert(Transaction *transaction, const KeyType &key, const ValueType &value) {
    HashTableDirectoryPage * dir_page = FetchDirectoryPage();
    int64_t split_bucket_index = KeyToDirectoryIndex(key, dir_page);
    uint32_t split_bucket_depth = dir_page->GetLocalDepth(split_bucket_index);
    if (split_bucket_depth == dir_page->GetGlobalDepth()) {
        dir_page->IncrGlobalDepth();
    }
    dir_page->IncrLocalDepth(split_bucket_index);
    page_id_t split_bucket_page_id = KeyToPageId(key, dir_page);
    HASH_TABLE_BUCKET_TYPE * split_bucket = FetchBucketPage(split_bucket_page_id);
    uint32_t origin_size = split_bucket->NumReadable();
    MappingType * origin_array = split_bucket->GetArrayCopy();
    split_bucket->Reset();
    page_id_t split_image_bucket_id;
    Page * page = buffer_pool_manager_->NewPage(&split_image_bucket_id);
    HASH_TABLE_BUCKET_TYPE * split_image_bucket = reinterpret_cast<HASH_TABLE_BUCKET_TYPE *>(page->GetData());
    uint32_t split_image_bucket_index = dir_page->GetSplitImageIndex(split_bucket_index);
    dir_page->SetLocalDepth(split_image_bucket_index, dir_page->GetLocalDepth(split_bucket_index));
    dir_page->SetBucketPageId(split_image_bucket_index, split_image_bucket_id);
    for (uint32_t i = 0; i < origin_size; i++) {
        MappingType temp = origin_array[i];
        uint32_t target_bucket_index = Hash(temp.first) & dir_page->GetLocalDepthMask(split_bucket_index);
        page_id_t target_page_id = dir_page->GetBucketPageId(target_bucket_index);
        if (target_page_id == split_bucket_page_id) {
            split_bucket->Insert(temp.first, temp.second, comparator_);
        } else {
            split_image_bucket->Insert(temp.first, temp.second, comparator_);
        }
    }
    delete [] origin_array;
    uint32_t diff = 1 << dir_page->GetLocalDepth(split_bucket_index);
    for (uint32_t i = split_bucket_index; i < dir_page->Size(); i += diff) {
        dir_page->SetBucketPageId(i, split_bucket_page_id);
        dir_page->SetLocalDepth(i, dir_page->GetLocalDepth(split_bucket_index));
    }
    for (uint32_t i = split_bucket_index; i >= diff; i -= diff) {
        dir_page->SetBucketPageId(i, split_bucket_page_id);
        dir_page->SetLocalDepth(i, dir_page->GetLocalDepth(split_bucket_index));
    }
    for (uint32_t i = split_image_bucket_index; i >= diff; i -= diff) {
        dir_page->SetBucketPageId(i, split_image_bucket_id);
        dir_page->SetLocalDepth(i, dir_page->GetLocalDepth(split_image_bucket_index));
    }
    for (uint32_t i = split_image_bucket_index; i < dir_page->Size(); i += diff) {
        dir_page->SetBucketPageId(i, split_image_bucket_id);
        dir_page->SetLocalDepth(i, dir_page->GetLocalDepth(split_image_bucket_index));
    }
    buffer_pool_manager_->UnpinPage(split_image_bucket_id, true);
    buffer_pool_manager_->UnpinPage(split_bucket_page_id, true);
    buffer_pool_manager_->UnpinPage(dir_page->GetPageId(), true);
    return Insert(transaction, key, value);
}

分裂时首先需要判断局部深度是否等于全局深度，如果相等的话就需要扩展全局深度之后再扩展局部深度，然后根据将要分裂的桶的索引获取分裂所得到的的新桶的索引，对元素的key重新进行hash运算确定将数据放入到哪一个桶中，最后再重新尝试插入新的元素。

template <typename KeyType, typename ValueType, typename KeyComparator>
bool HASH_TABLE_TYPE::Remove(Transaction *transaction, const KeyType &key, const ValueType &value) {
    HashTableDirectoryPage * dir_page = FetchDirectoryPage();
    page_id_t page_id = KeyToPageId(key, dir_page);
    HASH_TABLE_BUCKET_TYPE * bucket = FetchBucketPage(page_id);
    bool res = bucket->Remove(key, value, comparator_);
    buffer_pool_manager_->UnpinPage(page_id, true);
    buffer_pool_manager_->UnpinPage(dir_page->GetPageId(), false);
    if (bucket->IsEmpty()) {
        Merge(transaction, key, value);
    }
    return res;
}

remove函数和getValue元素相对简单，只需要通过key获取对应桶的索引，然后从桶中移除相应的数据即可。

template <typename KeyType, typename ValueType, typename KeyComparator>
void HASH_TABLE_TYPE::Merge(Transaction *transaction, const KeyType &key, const ValueType &value) {
    HashTableDirectoryPage * dir_page = FetchDirectoryPage();
    uint32_t target_bucket_index = KeyToDirectoryIndex(key, dir_page);
    page_id_t target_bucket_page_id = dir_page->GetBucketPageId(target_bucket_index);
    uint32_t target_image_index = dir_page->GetSplitImageIndex(target_bucket_index);
    uint32_t local_depth = dir_page->GetLocalDepth(target_bucket_index);
    if (local_depth == 0) {
        buffer_pool_manager_->UnpinPage(dir_page->GetPageId(), false);
        return;
    }
    if (local_depth != dir_page->GetLocalDepth(target_image_index)) {
        buffer_pool_manager_->UnpinPage(dir_page->GetPageId(), false);
        return;
    }
    HASH_TABLE_BUCKET_TYPE * target_bucket = FetchBucketPage(target_bucket_page_id);
    if (!target_bucket->IsEmpty()) {
        buffer_pool_manager_->UnpinPage(target_bucket_page_id, false);
        buffer_pool_manager_->UnpinPage(dir_page->GetPageId(), false);
        return;
    }
    buffer_pool_manager_->UnpinPage(target_bucket_page_id, false);
    buffer_pool_manager_->DeletePage(target_bucket_page_id);
    page_id_t  image_bucket_page_id = dir_page->GetBucketPageId(target_image_index);
    dir_page->SetBucketPageId(target_bucket_index, image_bucket_page_id);
    dir_page->DecrLocalDepth(target_bucket_index);
    dir_page->DecrLocalDepth(target_image_index);
    for (uint32_t i = 0; i < dir_page->Size(); i++) {
        if (dir_page->GetBucketPageId(i) == target_bucket_page_id || dir_page->GetBucketPageId(i) == image_bucket_page_id) {
            dir_page->SetBucketPageId(i, image_bucket_page_id);
            dir_page->SetLocalDepth(i, dir_page->GetLocalDepth(target_bucket_index));
        }
    }
    while (dir_page->CanShrink()) {
        dir_page->DecrGlobalDepth();
    }
    buffer_pool_manager_->UnpinPage(dir_page->GetPageId(), true);
}

桶合并时需要先判断这个桶能不能被合并，如果桶的局部深度为0，桶的局部深度不等于对应的image桶的深度，或者桶不为空，那么这个桶都不能被合并，然后将这个桶中原来所有的信息都设置为将要合并进的桶中的信息。
