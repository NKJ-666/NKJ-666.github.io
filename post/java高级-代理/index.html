Java高级 - 代理

什么是代理

代理类主要负责为委托类预处理消息、过滤消息、把消息转发给委托类，以及事后处理消息等。代理类与委托类之间通常会存在关联关系，一个代理类的对象与一个委托类的对象关联，代理类的对象本身并不真正实现服务，而是通过调用委托类的对象的相关方法，来提供特定的服务。

静态代理

public interface Sales {
    void sell();
}

public class Owner implements Sales{
    @Override
    public void sell() {
        System.out.println("我是卖主正在卖一些东西");
    }
}

public class Agents implements Sales{
    private Owner owner;

    @Override
    public void sell() {
        System.out.println("我是中介，正在核实买家信息");
        getOwner().sell();
        System.out.println("正在提取收成");
    }
    
    public Owner getOwner() {
    if(owner == null){
        owner = new Owner();
    }
    return owner;
	}
}

其中Agents就是一个代理类，其内部有一个用户类的对象，可以在Agents类中设置预处理消息、过滤消息、把消息转发给委托类，以及事后处理消息等。

然后测试

Sales sales = new Agents();
sales.sell();

但是这样有一个问题，代理对象需要与目标对象实现一样的接口,所以会有很多代理类,类太多.同时,一旦接口增加方法,目标对象与代理对象都要维护。这时就可以用动态代理了。

动态代理

动态代理是基于Java的反射机制生成动态代理类的。

动态代理中有一个重要的类(proxy)和一个重要的接口(InvocationHandler)

每一个动态代理类的调用处理程序都必须实现InvocationHandler接口，并且每个代理类的实例都关联到了实现该接口的动态代理类调用处理程序中，当我们通过动态代理对象调用一个方法时候，这个方法的调用就会被转发到实现InvocationHandler接口类的invoke方法来调用。

public interface Sales {	//代理接口
    void sell();

    void show();
}

public class SalesInvocationHandler implements InvocationHandler {
    private Object object;

    public SalesInvocationHandler(Object object){
        this.object = object;
    }

    @Override
    public Object invoke(Object proxy, Method method, Object[] args) throws Throwable {
        System.out.println("method start");

        //判断要调用的是哪个函数
        if (method.getName().equals("show")){
            method.invoke(object,args);
            System.out.println("show");
        }else{
            method.invoke(object,args);
            System.out.println("sell");
        }

        System.out.println("method end");
        return null;
    }
}

//测试
Sales sales = new Agents();
SalesInvocationHandler handler = new SalesInvocationHandler(sales);
Sales sales1 = (Sales) Proxy.newProxyInstance(sales.getClass().getClassLoader(),
        sales.getClass().getInterfaces(),
        handler);
sales1.sell();
